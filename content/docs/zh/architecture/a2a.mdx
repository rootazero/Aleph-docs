`a2a` 模块提供代理到代理（A2A）协议适配器，使 Aleph 代理能够相互通信并生成子代理以进行协作任务执行。

## 概览

A2A 是一种代理间通信协议，允许：
- **子代理生成** —— 创建子代理以并行执行任务
- **代理间消息传递** —— 在同一系统中的代理之间发送消息
- **任务委托** —— 将子任务委托给专门的代理
- **结果聚合** —— 收集和合并多个代理的结果

## 架构

```
A2A 适配器
  ├── 域 —— 核心 A2A 类型和消息
  ├── 端口 —— 协议接口定义
  ├── 服务 —— 代理协调的业务逻辑
  ├── 适配器 —— 具体协议实现
  ├── 配置 —— A2A 配置
  └── 子代理 —— 子代理生命周期管理
```

## 核心组件

### 域

A2A 消息的核心类型：

```rust
// src/a2a/domain/mod.rs
pub struct AgentMessage {
    pub from: AgentId,
    pub to: AgentId,
    pub payload: MessagePayload,
}

pub enum MessagePayload {
    TaskRequest { task: TaskSpec },
    TaskResponse { result: TaskResult },
    StatusUpdate { state: AgentState },
}
```

### 服务

编排代理间通信：

```rust
// src/a2a/service/mod.rs
pub struct A2AService {
    adapter: Box<dyn A2AAdapter>,
}

impl A2AService {
    pub async fn spawn_sub_agent(
        &self,
        spec: AgentSpec,
    ) -> Result<AgentId>;
    
    pub async fn send_message(
        &self,
        msg: AgentMessage,
    ) -> Result<()>;
    
    pub async fn collect_results(
        &self,
        task_id: TaskId,
    ) -> Result<Vec<TaskResult>>;
}
```

### 子代理

管理子代理生命周期：

```rust
// src/a2a/sub_agent/mod.rs
pub struct SubAgentManager {
    active: HashMap<AgentId, SubAgentHandle>,
}

impl SubAgentManager {
    pub async fn spawn(
        &mut self,
        spec: AgentSpec,
    ) -> Result<AgentId>;
    
    pub async fn terminate(
        &mut self,
        id: AgentId,
    ) -> Result<()>;
}
```

## 消息流

```
父代理                        A2A 服务                       子代理
     │                            │                            │
     │── spawn(spec) ────────────▶│                            │
     │                            │── create ─────────────────▶│
     │                            │◀─ AgentId ─────────────────│
     │◀─ AgentId ─────────────────│                            │
     │                            │                            │
     │── send(task) ─────────────▶│── forward ────────────────▶│
     │                            │                            │
     │                            │◀─ result ──────────────────│
     │◀─ TaskResult ──────────────│                            │
```

## 配置

A2A 通过 `config.toml` 配置：

```toml
[a2a]
enabled = true
max_sub_agents = 10
sub_agent_timeout = 300  # 秒
```
