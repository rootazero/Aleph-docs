---
title: "世界模型"
description: "用于主动代理行为、上下文推断和行动规划的世界模型。"
---

`worldmodel` 模块为守护进程的主动行为提供世界模型，使代理能够推断上下文、预测需求并自主规划行动。

## 概览

世界模型支持：
- **上下文推断** —— 从观察中推断用户意图和上下文
- **状态跟踪** —— 维护当前系统和用户状态的模型
- **行动预测** —— 预测可能需要哪些行动
- **主动规划** —— 为预期需求生成计划

## 架构

```
WorldModel
  ├── CoreState —— 当前系统状态
  ├── EnhancedContext —— 带有推断的丰富上下文
  ├── InferenceCache —— 缓存的推断结果
  └── PendingAction —— 排队的主动行动
```

## 核心组件

### WorldModel

中央世界模型外观：

```rust
// src/daemon/worldmodel/mod.rs
pub struct WorldModel {
    state: CoreState,
    inference_cache: InferenceCache,
    config: WorldModelConfig,
}

impl WorldModel {
    pub fn update_state(
        &mut self,
        event: SystemEvent,
    );
    
    pub fn infer_context(
        &self,
        raw_context: &Context,
    ) -> EnhancedContext;
    
    pub fn predict_actions(
        &self,
    ) -> Vec<PendingAction>;
    
    pub fn get_state(&self,
    ) -> &CoreState;
}
```

### CoreState

表示当前系统状态：

```rust
// src/daemon/worldmodel/mod.rs
pub struct CoreState {
    pub system_load: SystemLoad,
    pub active_sessions: Vec<SessionId>,
    pub recent_events: Vec<SystemEvent>,
    pub user_presence: PresenceStatus,
}
```

### EnhancedContext

带有推断的丰富上下文：

```rust
pub struct EnhancedContext {
    pub raw: Context,
    pub inferred_intent: Option<Intent>,
    pub predicted_needs: Vec<Need>,
    pub confidence: f32,
}
```

### PendingAction

排队执行的行动：

```rust
pub struct PendingAction {
    pub id: ActionId,
    pub action_type: ActionType,
    pub priority: Priority,
    pub scheduled_at: DateTime<Utc>,
    pub confidence: f32,
}
```

## 活动类型

世界模型跟踪不同的活动类型：

```rust
pub enum ActivityType {
    UserMessage,
    SystemEvent,
    ScheduledTask,
    ExternalTrigger,
}
```

## 与分派器集成

世界模型为主动分派器提供信息：

```
观察 → WorldModel.update_state() → infer_context()
                                              │
                                              ▼
                                       predict_actions()
                                              │
                                              ▼
                                       Dispatcher.dispatch()
```

## 配置

```toml
[daemon.worldmodel]
enabled = true
inference_timeout = 5000  # 毫秒
max_pending_actions = 20
confidence_threshold = 0.7
```
