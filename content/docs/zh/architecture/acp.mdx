`acp` 模块实现代理客户端协议（ACP），使 Aleph 能够将外部 CLI 工具作为一等代理适配器集成。

## 概览

ACP 管理外部 CLI 工具作为代理适配器，支持两种模式：
- **工具模式** —— LLM 分派的工具调用（只读操作）
- **代理模式** —— 具有完整读/写能力的直接对话

支持的适配器：
- Claude Code
- Codex (OpenAI)
- Gemini CLI

## 架构

```
ACP 管理器
  ├── 适配器 —— 协议抽象
  ├── 适配器 —— 具体实现
  │     ├── ClaudeCodeAdapter
  │     ├── CodexAdapter
  │     └── GeminiAdapter
  ├── 会话 —— ACP 会话生命周期
  ├── 传输 —— 通信层
  └── 协议 —— 消息协议定义
```

## 核心组件

### ACP 管理器

所有 ACP 会话的中央协调器：

```rust
// src/acp/manager.rs
pub struct AcpManager {
    sessions: HashMap<String, AcpSession>,
    persistence_hook: Option<PersistenceHook>,
}

impl AcpManager {
    pub async fn create_session(
        &mut self,
        harness_id: String,
        cwd: String,
        adapter_type: AdapterType,
    ) -> Result<String>;  // 返回 acp_session_id
    
    pub async fn send_message(
        &self,
        acp_session_id: String,
        message: String,
    ) -> Result<()>;
    
    pub async fn stream_response(
        &self,
        acp_session_id: String,
        callback: AcpChunkCallback,
    ) -> Result<()>;
}
```

### 适配器 Trait

所有 ACP 适配器的抽象接口：

```rust
// src/acp/adapter.rs
pub trait AcpAdapter: Send + Sync {
    fn name(&self) -> &str;
    fn is_available(&self) -> bool;
    async fn execute(
        &self, 
        cwd: &str, 
        args: &[String]
    ) -> Result<AcpOutput>;
}
```

### 会话

表示单个 ACP 对话：

```rust
// src/acp/session.rs
pub struct AcpSession {
    harness_id: String,
    acp_session_id: String,
    cwd: String,
    adapter: Box<dyn AcpAdapter>,
}
```

## 会话事件

ACP 管理器发出会话持久化事件：

```rust
pub enum AcpSessionEvent {
    Created { harness_id, acp_session_id, cwd },
    Updated { harness_id, acp_session_id },
    Removed { harness_id, cwd },
}
```

## 操作模式

### 工具模式

在工具模式中，LLM 通过 ACP 分派只读操作：

```
LLM ──> 工具调用 ──> ACP 适配器 ──> 外部 CLI
         (只读)       (claude code read)
```

### 代理模式

在代理模式中，用户直接与外部工具对话：

```
用户 ──> Aleph ──> ACP 管理器 ──> 外部 CLI
              (完整读/写)
```

## 配置

```toml
[acp]
enabled = true
default_adapter = "claude-code"

[acp.adapters.claude-code]
command = "claude"
args = ["--output", "json"]

[acp.adapters.codex]
command = "codex"
args = []
```
