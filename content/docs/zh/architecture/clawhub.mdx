> **已弃用 —— 历史参考。** ClawHub 不再是一等源。它的专属设置标签页已被移除，其发现职责被纳入[扩展商店](/docs/zh/concepts/extensions-store)，该商店以**扩展**为统一概念，按功能类别浏览技能、插件和 MCP 服务器。如今的一等源是 Claude/Aleph 插件市场、Official MCP Registry 和 Docker MCP Catalog。ClawHub 仅可通过商店智能体的长尾路径访问（智能体读取某个 URL/仓库，以安装任何已编排源都未覆盖的扩展）。以下架构作为 `clawhub` 模块的历史参考予以保留。

`clawhub` 模块提供与 [ClawHub](https://clawhub.ai) 的 HTTP 客户端集成，ClawHub 是一个用于发现和安装社区贡献技能技能注册表。

## 概览

ClawHub 集成使 Aleph 能够：
- **浏览技能** —— 搜索和发现社区技能
- **安装技能** —— 从注册表下载和安装技能
- **同步元数据** —— 保持技能元数据最新

## 架构

```
ClawHubClient
  ├── HTTP 客户端 (reqwest)
  ├── 类型 —— 请求/响应模式
  └── 认证 (API token)
```

## 核心组件

### ClawHubClient

ClawHub API 的 HTTP 客户端：

```rust
// src/clawhub/client.rs
pub struct ClawHubClient {
    base_url: String,
    client: reqwest::Client,
    auth_token: Option<String>,
}

impl ClawHubClient {
    pub fn new(base_url: String) -> Self;
    
    pub async fn search_skills(
        &self,
        query: &str,
    ) -> Result<Vec<SkillListing>>;
    
    pub async fn get_skill(
        &self,
        id: &str,
    ) -> Result<SkillDetail>;
    
    pub async fn download_skill(
        &self,
        id: &str,
        dest: &Path,
    ) -> Result<()>;
}
```

### 类型

```rust
// src/clawhub/types.rs
pub struct SkillListing {
    pub id: String,
    pub name: String,
    pub description: String,
    pub author: String,
    pub version: String,
    pub downloads: u64,
    pub rating: f32,
}

pub struct SkillDetail {
    pub listing: SkillListing,
    pub readme: String,
    pub dependencies: Vec<String>,
    pub tools: Vec<String>,
}
```

## 用法

```rust
use alephcore::clawhub::ClawHubClient;

let client = ClawHubClient::new(
    "https://api.clawhub.ai".to_string()
);

// 搜索技能
let results = client.search_skills("code review").await?;

// 下载技能
client.download_skill("code-review", 
    Path::new("~/.aleph/skills/")
).await?;
```

## 配置

```toml
[clawhub]
enabled = true
base_url = "https://api.clawhub.ai"
api_token = "${CLAWHUB_TOKEN}"
```
