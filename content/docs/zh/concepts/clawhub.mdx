`clawhub` 模块提供用于与 ClawHub 技能注册表交互的 HTTP 客户端和类型定义。ClawHub 是一个公共仓库，用户可以在其中浏览、搜索和下载用于 Aleph 和兼容代理的技能。

## 设计理念

ClawHub 集成遵循两个原则：

1. **只读访问** —— 客户端从注册表获取技能；它不上传或修改注册表内容。公共 API 访问不需要认证。
2. **干净的类型分离** —— 原始 API 类型通过 `From` 实现转换为内部统一类型，保持 HTTP 层与领域逻辑解耦。

## 核心组件

### ClawHubClient

注册表操作的主 HTTP 客户端：

```rust
pub struct ClawHubClient {
    base_url: String,
    client: reqwest::Client,
}

impl ClawHubClient {
    pub fn new() -> Self { /* ... */ }
    
    pub async fn browse(
        &self,
        category: Option<String>,
    ) -> Result<Vec<SkillSummary>> { /* ... */ }
    
    pub async fn search(
        &self,
        query: &str,
    ) -> Result<Vec<SkillSummary>> { /* ... */ }
    
    pub async fn download(
        &self,
        slug: &str,
    ) -> Result<Vec<u8>> { /* ... */ }
}
```

**关键行为：**
- 当浏览端点返回空时，浏览回退到搜索
- 下载在临时文件名中使用之前清理 slug（将 `/` 替换为 `-`）
- 所有方法使用 `?` 或 `map_err` 进行错误传播

### 类型

核心注册表类型：

```rust
pub struct SkillSummary {
    pub slug: String,           // "owner/skill" 格式
    pub name: String,
    pub description: String,
    pub version: String,
    pub downloads: u64,
}

pub struct SkillDetail {
    pub summary: SkillSummary,
    pub readme: String,
    pub manifest: SkillManifest,
}
```

## API 操作

### 浏览

按类别列出技能。如果类别端点返回空，则回退到搜索：

```rust
let client = ClawHubClient::new();
let skills = client.browse(Some("productivity".to_string())).await?;
```

### 搜索

跨注册表的全文搜索：

```rust
let results = client.search("file organizer").await?;
```

### 下载

获取技能包作为 ZIP 存档：

```rust
let zip_bytes = client.download("aleph/file-organizer").await?;
// slug "aleph/file-organizer" 变为临时文件 "clawhub-aleph-file-organizer-{uuid}.zip"
```

## 工具函数

### URL 编码

```rust
pub fn encode_slug_path(slug: &str) -> String
```

对 slug 进行编码以用于 URL，使用每段百分比编码。

### 版本比较

```rust
pub fn is_newer_version(current: &str, latest: &str) -> bool
```

比较两个语义版本字符串。对于非 semver 输入回退到字符串比较。

### 时间戳转换

```rust
pub fn unix_ms_to_rfc3339(ts: i64) -> String
```

将 Unix 毫秒转换为 RFC 3339 格式。对于无效时间戳返回空字符串。

---

## 错误处理

客户端使用 `reqwest` 进行 HTTP，并从所有操作返回 `Result<T, ClawHubError>`。错误包括：
- 网络故障
- HTTP 非 2xx 状态码
- 无效响应解析
- 下载期间的文件系统错误
