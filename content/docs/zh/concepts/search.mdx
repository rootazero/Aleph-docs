`search` 模块为 Aleph 提供实时 Web 搜索，使代理能够访问其训练数据截止日期之外的当前信息。它通过统一的 trait 接口支持多个搜索后端。

## 设计理念

搜索系统遵循三个原则：

1. **提供商抽象** — 所有搜索后端实现相同的 `SearchProvider` trait
2. **隐私优先** — 支持自托管的 SearXNG 以及商业提供商
3. **故障转移路由** — 注册表尝试提供商直到一个成功

## 核心类型

### SearchResult

所有提供商的统一结果结构：

```rust
pub struct SearchResult {
    pub title: String,
    pub url: String,
    pub snippet: String,
    pub source: String,      // 提供商名称
}
```

### SearchOptions

搜索行为的配置：

```rust
pub struct SearchOptions {
    pub max_results: usize,
    pub include_images: bool,
    pub time_range: Option<TimeRange>,
    pub safe_search: bool,
}
```

### SearchProvider Trait

```rust
#[async_trait]
pub trait SearchProvider: Send + Sync {
    async fn search(
        &self,
        query: &str,
        options: &SearchOptions,
    ) -> Result<Vec<SearchResult>>;

    fn name(&self) -> &str;
    fn test_connection(&self,
    ) -> Pin<Box<dyn Future<Output = ProviderTestResult> + Send>>;
}
```

## SearchRegistry

管理具有故障转移的多个提供商：

```rust
pub struct SearchRegistry {
    providers: Vec<Box<dyn SearchProvider>>,
}

impl SearchRegistry {
    pub async fn search(
        &self,
        query: &str,
        options: &SearchOptions,
    ) -> Result<Vec<SearchResult>> {
        // 按顺序尝试每个提供商直到一个成功
        for provider in &self.providers {
            match provider.search(query, options).await {
                Ok(results) => return Ok(results),
                Err(e) => {
                    tracing::warn!(
                        provider = provider.name(),
                        error = %e,
                        "Search provider failed, trying next"
                    );
                }
            }
        }
        Err(SearchError::AllProvidersFailed)
    }
}
```

**隐私说明：** 用户查询不包含在错误消息中，以防止敏感搜索词的意外记录。

## 提供商测试

```rust
pub struct SearchProviderTestConfig {
    pub provider_type: String,  // "tavily", "brave", "searxng" 等
    pub api_key: Option<String>,
    pub base_url: Option<String>,      // SearXNG 必需
    pub engine_id: Option<String>,    // Google CSE 必需

```

## 使用示例

```rust
use alephcore::search::{SearchProvider, SearchOptions};
use alephcore::search::providers::TavilyProvider;

let provider = TavilyProvider::new("tvly-xxx".to_string())?;
let options = SearchOptions::default();

let results = provider.search("Rust programming language", &options).await?;

for result in results {
    println!("Title: {}", result.title);
    println!("URL: {}", result.url);
    println!("Snippet: {}\n", result.snippet);
}
```

## 安全属性

- **无查询泄漏** — 错误消息使用固定字符串，而非用户查询
- **安全截断** — 延迟使用 `.min(u32::MAX as u128)` 进行饱和
- **无重复头** — `reqwest` 自动处理 `Content-Type`
- **无锁问题** — 使用 `unwrap_or_else(|e| e.into_inner())`
- **无 SQL 注入** — 无数据库查询
