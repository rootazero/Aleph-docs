技能方法管理技能系统 v2——一个统一的技能发现、注册和生命周期管理框架。技能为代理添加工具、提示和能力。

> **注意：** `skills.*` 是统一 `extensions.*` 门面背后按类型划分的后端之一。对于日常用户，技能、插件和 MCP 服务器统一在同一个**扩展**概念下，通过[扩展商店](/docs/zh/concepts/extensions-store)浏览和安装。下面的 `skills.*` 方法仍是底层的技能后端；伞形 RPC 参见 [extensions.*](/docs/zh/gateway/methods/extensions)。

## 方法

### skills.status

返回所有已注册技能的完整状态，包括其启用状态、范围和依赖项。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "skills.status",
  "params": {}
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "skills": [
      {
        "id": "builtin:shell",
        "name": "Shell Execution",
        "enabled": true,
        "scope": "tool",
        "version": "1.0.0",
        "dependencies": []
      },
      {
        "id": "user:web_search",
        "name": "Web Search",
        "enabled": true,
        "scope": "tool",
        "version": "2.1.0",
        "dependencies": ["builtin:http"]
      }
    ]
  }
}
```

### skills.update

更新技能的配置（启用状态和/或提示范围）。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "skills.update",
  "params": {
    "skill_id": "user:web_search",
    "enabled": false,
    "scope": "disabled"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "skill": {
      "id": "user:web_search",
      "name": "Web Search",
      "enabled": false,
      "scope": "disabled",
      "version": "2.1.0"
    }
  }
}
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `skill_id` | string | 是 | 要更新的技能 ID |
| `enabled` | boolean | 否 | 启用或禁用技能 |
| `scope` | string | 否 | 提示范围：`"system"`、`"tool"`、`"standalone"`、`"disabled"` |

**范围值：**

| 范围 | 说明 |
|-------|-------------|
| `system` | 注入到系统提示中 |
| `tool` | 可作为可调用工具使用 |
| `standalone` | 独立于代理使用 |
| `disabled` | 未加载 |

### skills.install_dep

为技能安装依赖项（如技能所需的系统包）。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "skills.install_dep",
  "params": {
    "skill_id": "user:web_search",
    "spec_id": "curl"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "result": {
    "result": "installed",
    "skill": {
      "id": "user:web_search",
      "name": "Web Search",
      "enabled": true
    }
  }
}
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `skill_id` | string | 是 | 技能 ID |
| `spec_id` | string | 否 | 依赖规范 ID（如包名） |

### skills.remove

从系统中移除技能。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "method": "skills.remove",
  "params": {
    "skill_id": "user:web_search"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "result": {
    "ok": true
  }
}
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `skill_id` | string | 是 | 要移除的技能 ID |

### skills.install

从 markdown 技能文件或目录安装技能。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "method": "skills.install",
  "params": {
    "path": "~/.aleph/skills/custom_skill.md"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "result": {
    "installed": true,
    "skill_id": "user:custom_skill"
  }
}
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `path` | string | 是 | 技能文件或目录的路径 |

## 技能目录

技能从以下目录加载（按顺序）：

1. `~/.aleph/skills/` — 用户安装的技能
2. `./skills/` — 项目本地技能
3. 与 Aleph 捆绑的内置技能
