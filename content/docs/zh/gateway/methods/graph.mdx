图谱方法管理知识图谱——一个实体、关系和观察的语义网络，代表从对话和文档中提取的结构化知识。

## 方法

### graph.query

使用自然语言或结构化查询查询知识图谱。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "graph.query",
  "params": {
    "query": "What projects is Alice working on?",
    "limit": 10
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "results": [
      {
        "entity": "Project Alpha",
        "relation": "worked_on_by",
        "target": "Alice",
        "confidence": 0.95
      }
    ]
  }
}
```

### graph.entities

列出知识图谱中的实体。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "graph.entities",
  "params": {
    "type": "person",
    "limit": 50
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "entities": [
      {
        "id": "ent-1",
        "name": "Alice",
        "type": "person",
        "observations": ["Software engineer", "Likes Rust"]
      }
    ]
  }
}
```

### graph.add_entity

向知识图谱添加新实体。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "graph.add_entity",
  "params": {
    "name": "Project Beta",
    "type": "project",
    "observations": ["AI documentation tool", "Built with Rust"]
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "result": {
    "entity_id": "ent-2",
    "added": true
  }
}
```

### graph.add_relation

在两个实体之间添加关系。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "method": "graph.add_relation",
  "params": {
    "from_entity": "Alice",
    "relation": "works_on",
    "to_entity": "Project Beta"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "result": {
    "relation_id": "rel-1",
    "added": true
  }
}
```

### graph.delete_entity

删除实体及其关系。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "method": "graph.delete_entity",
  "params": {
    "entity_id": "ent-2"
  }
}
```
