引导式设置流程的配置向导。

:::warning
本页记录的是预期 API。`wizard.*` 方法（`wizard.start`、`wizard.answer`、`wizard.next`、`wizard.cancel`、`wizard.status`）在源处理程序（`src/gateway/handlers/wizard.rs`）中定义，但**当前未在运行时处理程序注册中接线**。它们反映的是设计目标，但尚未通过 JSON-RPC 公开。请参阅[方法参考](/docs/zh/gateway/methods)获取准确的方法列表。
:::

## 概述

向导系统提供多步配置体验，包含：
- 多种步骤类型（选择、文本、确认等）
- 基于会话的状态管理
- 支持多个并发向导

## 方法

### wizard.start

启动新的向导会话。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "wizard.start",
  "params": {
    "wizard_type": "onboarding",
    "initial_data": {}
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "session_id": "wizard-uuid",
    "step": {
      "id": "step-1",
      "type": "note",
      "title": "Welcome",
      "message": "Welcome to Aleph setup!"
    },
    "status": "running"
  }
}
```

### wizard.answer

回答当前步骤并获取下一步。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "wizard.answer",
  "params": {
    "session_id": "wizard-uuid",
    "step_id": "step-2",
    "value": "anthropic"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "done": false,
    "step": {
      "id": "step-3",
      "type": "text",
      "message": "Enter your API key:",
      "placeholder": "sk-...",
      "sensitive": true
    },
    "status": "running"
  }
}
```

### wizard.next

无需回答即可获取下一步（用于注释/介绍）。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "wizard.next",
  "params": {
    "session_id": "wizard-uuid"
  }
}
```

### wizard.cancel

取消活动向导会话。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "method": "wizard.cancel",
  "params": {
    "session_id": "wizard-uuid"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "result": {
    "cancelled": true
  }
}
```

### wizard.status

获取向导会话的状态。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "method": "wizard.status",
  "params": {
    "session_id": "wizard-uuid"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "result": {
    "status": "running"
  }
}
```

## 步骤类型

| 类型 | 说明 | 值类型 |
|------|-------------|------------|
| `note` | 信息性消息 | 无 |
| `select` | 单选 | String |
| `multi_select` | 多选 | String[] |
| `text` | 文本输入 | String |
| `confirm` | 是/否问题 | Boolean |
| `progress` | 进度指示器 | 无 |

## 步骤结构

```json
{
  "id": "step-1",
  "type": "select",
  "title": "Provider Selection",
  "message": "Which AI provider would you like to use?",
  "options": [
    {
      "value": "anthropic",
      "label": "Anthropic (Claude)",
      "hint": "Recommended for coding"
    },
    {
      "value": "openai",
      "label": "OpenAI (GPT)",
      "hint": "GPT-4o and o3 series"
    }
  ],
  "initial_value": "anthropic",
  "sensitive": false
}
```

## 可用向导

| 类型 | 说明 |
|------|-------------|
| `onboarding` | 完整的 10 阶段设置 |
| `quick-setup` | 最小化配置 |
| `provider-setup` | 添加新提供商 |

## 向导状态

| 状态 | 说明 |
|--------|-------------|
| `running` | 向导处于活动状态 |
| `done` | 成功完成 |
| `cancelled` | 用户已取消 |
| `error` | 发生错误 |

## 入门流程阶段

1. **欢迎** - 介绍
2. **提供商选择** - 选择 AI 提供商
3. **凭据** - 输入 API 密钥
4. **主模型** - 选择主模型
5. **辅助模型** - 可选故障转移
6. **思考级别** - 配置 AI 深度
7. **消息应用** - 可选集成
8. **审查** - 配置摘要
9. **完成** - 应用配置
10. **完成** - 成功消息
