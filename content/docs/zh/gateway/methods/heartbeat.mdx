心跳方法管理健康检查和心跳信号，用于监控系统健康、连接性和组件状态。

## 方法

### heartbeat.ping

简单的 ping/pong 健康检查。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "heartbeat.ping",
  "params": {}
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "pong": true,
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```

### heartbeat.status

获取所有组件的详细心跳状态。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "heartbeat.status",
  "params": {}
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "healthy": true,
    "components": [
      {
        "name": "gateway",
        "status": "healthy",
        "last_beat": "2024-01-15T10:30:00Z",
        "latency_ms": 5
      },
      {
        "name": "memory",
        "status": "healthy",
        "last_beat": "2024-01-15T10:29:58Z",
        "latency_ms": 12
      }
    ]
  }
}
```

### heartbeat.check

检查特定组件的健康状况。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "heartbeat.check",
  "params": {
    "component": "mcp"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "result": {
    "component": "mcp",
    "healthy": true,
    "last_beat": "2024-01-15T10:30:00Z"
  }
}
```

## 健康状态

| 状态 | 说明 |
|-------|-------------|
| `healthy` | 组件正常运行 |
| `degraded` | 组件运行缓慢或部分故障 |
| `unhealthy` | 组件已故障 |
| `unknown` | 最近未收到心跳 |
