title: "mcp.*"
description: "MCP 服务器管理 RPC 方法"
---

MCP（模型上下文协议）方法管理外部 MCP 服务器——添加、更新、删除、启动、停止和查询通过 MCP 协议公开工具、资源和提示的服务器的能力。

:::note
`mcp.*` 是统一 `extensions.*` 门面背后的按类型后端之一。对于日常用户，MCP 服务器、插件和技能统一为一个概念——**扩展**（类型为 `Mcp`）。要按功能类别浏览和安装扩展，请参阅 [extensions.* 方法](/docs/zh/gateway/methods/extensions) 和 [扩展商店](/docs/zh/concepts/extensions-store)。下方的 `mcp.*` 方法仍是 MCP 插件宿主的底层控制面。
:::

## 方法

### mcp.list

列出所有配置的 MCP 服务器及其状态。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "mcp.list",
  "params": {}
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "servers": [
      {
        "id": "filesystem",
        "name": "File System",
        "transport": "stdio",
        "status": "running",
        "healthy": true,
        "tools_count": 5,
        "resources_count": 0,
        "prompts_count": 0
      }
    ]
  }
}
```

### mcp.add

添加新的 MCP 服务器配置。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "mcp.add",
  "params": {
    "config": {
      "id": "filesystem",
      "name": "File System",
      "transport": "stdio",
      "command": "npx",
      "args": ["@modelcontextprotocol/server-filesystem", "/home/user"],
      "requires_runtime": "node"
    }
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "ok": true
  }
}
```

**配置字段：**

| 字段 | 类型 | 必填 | 说明 |
|-------|------|----------|-------------|
| `id` | string | 是 | 唯一服务器标识符 |
| `name` | string | 是 | 显示名称 |
| `transport` | string | 是 | `"stdio"` 或 `"http"` |
| `command` | string | 否 | 要运行的命令（用于 stdio） |
| `args` | string[] | 否 | 命令参数 |
| `url` | string | 否 | HTTP URL（用于 http 传输） |
| `env` | object | 否 | 环境变量 |
| `requires_runtime` | string | 否 | 运行时要求（如 `"node"`） |

### mcp.update

更新现有的 MCP 服务器配置（使用 upsert 语义）。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "mcp.update",
  "params": {
    "config": {
      "id": "filesystem",
      "name": "File System Updated",
      "args": ["@modelcontextprotocol/server-filesystem", "/home/user", "/tmp"]
    }
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "result": {
    "ok": true
  }
}
```

### mcp.list_pending_approvals

列出所有待处理的 MCP 审批请求，等待用户响应。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 12,
  "method": "mcp.list_pending_approvals",
  "params": {}
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 12,
  "result": []
}
```

:::note
此方法当前返回空数组，因为审批处理程序集成待完成。
:::

### mcp.respond_approval

提交用户对 MCP 审批请求的响应。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 13,
  "method": "mcp.respond_approval",
  "params": {
    "request_id": "req-123",
    "approved": true,
    "reason": "Looks safe"
  }
}
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `request_id` | string | 是 | 审批请求 ID |
| `approved` | boolean | 是 | 是否批准 |
| `reason` | string | 否 | 决策的可选原因 |

### mcp.cancel_approval

取消待处理的 MCP 审批请求。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 14,
  "method": "mcp.cancel_approval",
  "params": {
    "request_id": "req-123"
  }
}
```
