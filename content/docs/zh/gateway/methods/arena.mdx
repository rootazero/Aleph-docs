竞技场方法管理评估竞技场——一个用于对 AI 模型进行基准测试、比较输出和针对测试数据集运行评估任务的系统。

## 方法

### arena.list

列出可用的评估任务和基准测试。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "arena.list",
  "params": {}
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "benchmarks": [
      {
        "id": "coding-bench",
        "name": "Coding Benchmark",
        "description": "Tests code generation and debugging",
        "tasks_count": 50
      }
    ]
  }
}
```

### arena.run

运行评估基准测试。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "arena.run",
  "params": {
    "benchmark_id": "coding-bench",
    "model": "claude-sonnet-4-20250514",
    "parallel": 4
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "run_id": "arena-run-1",
    "status": "running",
    "progress": {
      "completed": 0,
      "total": 50
    }
  }
}
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `benchmark_id` | string | 是 | 要运行的基准测试 |
| `model` | string | 是* | 要评估的模型（如果基准测试支持模型选择） |
| `parallel` | number | 否 | 并行任务数 |

### arena.status

获取正在运行的评估的状态。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "arena.status",
  "params": {
    "run_id": "arena-run-1"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "result": {
    "run_id": "arena-run-1",
    "status": "completed",
    "progress": {
      "completed": 50,
      "total": 50
    },
    "results": {
      "accuracy": 0.92,
      "avg_latency_ms": 1200
    }
  }
}
```

### arena.results

获取已完成评估的详细结果。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "method": "arena.results",
  "params": {
    "run_id": "arena-run-1"
  }
}
```

### arena.compare

比较多个模型运行的结果。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "method": "arena.compare",
  "params": {
    "run_ids": ["arena-run-1", "arena-run-2"]
  }
}
```
