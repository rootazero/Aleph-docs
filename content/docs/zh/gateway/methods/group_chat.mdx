群聊方法管理一个多智能体群聊**会话**——在共享对话中，多个智能体（以及用户）按归属轮流发言。

> 面板三栏式群聊窗口中显示的持久化按团队聊天线程使用 `teams.chat.*` 方法（`teams.chat.send`、`teams.chat.thread`、`teams.chat.history`）——参见 [teams.*](/docs/zh/gateway/methods/teams)。下面的 `group_chat.*` 方法是那些团队线程所基于的更底层的多智能体群聊**会话原语**。

## 方法

### group_chat.start

在一组智能体之上启动一个新的群聊会话。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "group_chat.start",
  "params": {
    "agents": ["planner", "coder", "reviewer"],
    "topic": "Plan the v2 migration"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "session_id": "gc-1",
    "agents": ["planner", "coder", "reviewer"],
    "created_at": "2024-01-15T10:30:00Z"
  }
}
```

### group_chat.continue

用一条新消息推进会话，并让各智能体依次响应。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "group_chat.continue",
  "params": {
    "session_id": "gc-1",
    "message": "What do you think about this approach?"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "messages": [
      { "agent_id": "planner", "display_name": "Planner", "text": "This looks solid. I suggest..." },
      { "agent_id": "coder", "display_name": "Coder", "text": "Implementation-wise, we could..." }
    ]
  }
}
```

### group_chat.mention

在会话中指名某个智能体（或所有人）——面板的 `@` 名册自动补全映射到此方法。使用 `@all` 来指名全部参与者。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "group_chat.mention",
  "params": {
    "session_id": "gc-1",
    "mention": "coder",
    "message": "Can you sketch the schema change?"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "result": {
    "messages": [
      { "agent_id": "coder", "display_name": "Coder", "text": "Here's the migration outline..." }
    ]
  }
}
```

### group_chat.history

获取某个会话的消息历史（用于驱动归属气泡的回放）。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "method": "group_chat.history",
  "params": {
    "session_id": "gc-1",
    "limit": 50
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "result": {
    "messages": [
      { "agent_id": "user", "display_name": "You", "text": "Plan the v2 migration" },
      { "agent_id": "planner", "display_name": "Planner", "text": "This looks solid. I suggest..." }
    ]
  }
}
```

### group_chat.list

列出活动的群聊会话。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "method": "group_chat.list",
  "params": {}
}
```

### group_chat.end

结束一个群聊会话。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 6,
  "method": "group_chat.end",
  "params": {
    "session_id": "gc-1"
  }
}
```
