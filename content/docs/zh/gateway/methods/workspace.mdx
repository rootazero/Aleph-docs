工作空间方法管理项目工作空间——隔离的环境，包含特定任务或项目的文件、配置和上下文。

## 方法

### workspace.list

列出所有工作空间。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "workspace.list",
  "params": {}
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "workspaces": [
      {
        "id": "ws-1",
        "name": "My Project",
        "path": "/home/user/projects/my-project",
        "created_at": "2024-01-15T10:30:00Z",
        "active": true
      }
    ]
  }
}
```

### workspace.create

创建新工作空间。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "workspace.create",
  "params": {
    "name": "New Project",
    "path": "/home/user/projects/new-project"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "workspace": {
      "id": "ws-2",
      "name": "New Project",
      "path": "/home/user/projects/new-project",
      "created_at": "2024-01-15T10:30:00Z",
      "active": false
    }
  }
}
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `name` | string | 是 | 工作空间显示名称 |
| `path` | string | 是 | 工作空间的目录路径 |

### workspace.get

获取特定工作空间的详细信息。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "workspace.get",
  "params": {
    "workspace_id": "ws-1"
  }
}
```

### workspace.activate

激活工作空间（将其设置为当前活动工作空间）。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "method": "workspace.activate",
  "params": {
    "workspace_id": "ws-1"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "result": {
    "activated": true
  }
}
```

### workspace.delete

删除工作空间。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "method": "workspace.delete",
  "params": {
    "workspace_id": "ws-2"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 5,
  "result": {
    "deleted": true
  }
}
```

:::warning
删除工作空间仅移除工作空间元数据。底层目录**不会**被删除。
:::

### workspace.files

列出工作空间中的文件。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 6,
  "method": "workspace.files",
  "params": {
    "workspace_id": "ws-1",
    "path": "/"
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 6,
  "result": {
    "files": [
      {
        "name": "src",
        "type": "directory",
        "path": "/src"
      },
      {
        "name": "README.md",
        "type": "file",
        "path": "/README.md",
        "size": 1024
      }
    ]
  }
}
```
