title: "identity.*"

---

身份方法管理会话期间 AI 的身份（灵魂）。获取有效身份、设置会话级覆盖和列出可用身份源。

## 方法

### identity.get

返回当前会话的有效 SoulManifest，包括会话覆盖是否处于活动状态。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "method": "identity.get"
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 1,
  "result": {
    "soul": {
      "identity": "I am Aleph, your AI assistant.",
      "directives": ["Be helpful", "Be concise"]
    },
    "has_session_override": false
  }
}
```

### identity.set

设置会话级身份覆盖。这会临时更改当前会话的 AI 行为，而无需修改全局身份文件。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "method": "identity.set",
  "params": {
    "soul": {
      "identity": "I am a specialized coding assistant.",
      "directives": ["Focus on code quality", "Explain concisely"]
    }
  }
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 2,
  "result": {
    "success": true
  }
}
```

**参数：**

| 参数 | 类型 | 必填 | 说明 |
|-----------|------|----------|-------------|
| `soul` | object | 是 | 要应用的 SoulManifest |
| `soul.identity` | string | 是 | 身份声明 |
| `soul.directives` | string[] | 否 | 行为指令 |

### identity.clear

清除会话级身份覆盖，恢复到全局身份。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "method": "identity.clear"
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 3,
  "result": {
    "success": true,
    "had_override": true
  }
}
```

### identity.list

列出所有可用身份源及其加载状态。

**请求：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "method": "identity.list"
}
```

**响应：**
```json
{
  "jsonrpc": "2.0",
  "id": 4,
  "result": {
    "sources": [
      {
        "source_type": "global",
        "path": "/Users/user/.aleph/soul.md",
        "loaded": true
      },
      {
        "source_type": "session",
        "path": "<session>",
        "loaded": true
      }
    ]
  }
}
```

**源类型：**

| 类型 | 说明 |
|------|-------------|
| `global` | 从 `~/.aleph/soul.md` 加载 |
| `session` | 会话级覆盖（通过 `identity.set` 设置） |

## 身份文件

全局身份从 `~/.aleph/soul.md` 加载。这是一个包含以下内容的 markdown 文件：

```markdown
# Soul

I am Aleph, a polymorphic AI assistant.

## Directives

- Be helpful and accurate
- Be concise unless asked for detail
- Respect user privacy
```
