# Terminology

This page establishes canonical names for Aleph concepts to ensure consistency across documentation, code, and UI.

> **Panel top-level modes** (йЎ¶еұӮжЁЎејҸ): the desktop panel has seven, in nav order вҖ” Chat (`/chat`), Dashboard (`/dashboard`), Memory (`/memory`), Agents (`/agents`), Teams (`/teams`), Extensions (`/extensions`), Settings (`/settings`).
